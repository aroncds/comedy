# Comedy

This is a simple project, you can post a joke in the blockchain(your joke can't be removed) and the user can buy tokens(LGH) and give to you, this is a way of thank for the laugh!

I hope you enjoy! =]

### Install
```sh
npm install
```

### Migrate:
```sh
truffle migrate
```

### Test:
```sh
truffle test
```

### Run:
```sh
npm start
```
