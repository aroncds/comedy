# Comedy

This project is a kidding, it isn't revolutionary, but is cool, here you can make a joke and post in the blockchain(your joke can't be removed), the users can buy tokens(LGH) and give to you for joke!

I hope you enjoy! =]

### Install
```sh
npm install
```

### Migrate:
```sh
truffle migrate
```

### Test:
```sh
truffle test
```

### Run:
```sh
npm start
```
